# -Movie-Search-System-Python-GUI-
A GUI-based Python project that simulates a movie recommendation system using graph algorithms.
## Features
- Genre-based movie system
- BFS search algorithm
- A* search algorithm
- Step-by-step algorithm visualization (logs)
- GUI using CustomTkinter

## Concepts Used
- Graph theory
- BFS algorithm
- A* algorithm
- Heuristic function
- GUI development

## How it works
Genres are connected as a graph. The system finds the best path from a starting genre to a target genre using BFS or A* and then recommends movies.

## Technologies
- Python
- CustomTkinter
